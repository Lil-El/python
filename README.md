# Python Learn

## 数据类型

不可变数据（3 个）：Number（数字）、String（字符串）、Tuple（元组）；

可变数据（3 个）：List（列表）、Dictionary（字典）、Set（集合）。

1. 数字

- int - 整数类型,如 42, -7
- float - 浮点数类型,如 3.14, -0.5
- complex - 复数类型,如 3+4j
- bool - 布尔类型,是 int 的子类,值为 True 或 False

2. 字符串

- str - 字符串类型,如 "hello", 'world'

3. 列表

- list - 列表类型,如 [1, 2, 3], ['a', 'b']

4. 元组

- tuple - 元组类型,如 (1, 2, 3), ('a', 'b')

5. 字典

- dict - 字典类型,如 {'name': 'Alice', 'age': 25}

6. 集合

- set - 集合类型,如 {1, 2, 3},无序且不重复
- frozenset - 不可变集合类型
